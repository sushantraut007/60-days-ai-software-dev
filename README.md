## How it works

1. GitHub Actions runs at **9 AM UTC** every day (free)
2. `calculate_day.py` determines today's project number
3. `generate_project.py` calls **GitHub Models** (gpt-4o, free) to generate everything
4. Project folder committed and pushed automatically
5. This README updates with live progress

**Zero cost** — uses only `GITHUB_TOKEN` (built into every repo).

## Setup in 3 steps

```bash
# Step 1 — Fork this repo on GitHub

# Step 2 — Set your start date (Settings → Variables → Actions):

# Step 3 — Allow write access (Settings → Actions → Workflow permissions):

# Actions → Daily AI Project Deploy → Run workflow
```
